# Шаблон с параметрами для техподдержки

Этот шаблон предназначен для генерации промпта, который затем можно отправить в LLM для создания ответа пользователю от имени службы поддержки.

## Параметры шаблона

- `user_name` - имя пользователя
- `product_name` - название продукта или сервиса
- `issue_description` - описание проблемы пользователя
- `tone` - желаемый тон ответа

## Что делает шаблон

Шаблон задает модели:
- роль специалиста поддержки
- контекст обращения
- цель ответа
- ограничения по стилю и длине
- ожидаемый формат результата

## Демонстрация

В файле `prompt_template.py` показаны 3 разных набора параметров, чтобы продемонстрировать гибкость шаблона.
